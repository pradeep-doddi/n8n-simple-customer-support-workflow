# n8n-simple-customer-support-workflow
A simple customer support workflow consisting a simple linear workflow. It simply receive a submission from the support form and it adds a data to a excel sheet and directly sends to message to customer email 
used a 3 step simple customer support automation which consists of filling a form,collecting user data,support to customer via email
